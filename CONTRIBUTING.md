# 貢獻指南

感謝您有興趣為 **Cloud-Assets-Template** 做出貢獻！這是一個開源項目，我們歡迎任何形式的改進與建議。

## 如何參與貢獻

1.  **Fork 本專案**：將專案 Fork 到您的 GitHub 帳號。
2.  **建立分支**：為您的修改建立一個新的分支 (`git checkout -b feature/AmazingFeature`)。
3.  **提交修改**：提交您的變更 (`git commit -m 'Add some AmazingFeature'`)。
4.  **推送到分支**：將變更推送到您的 GitHub (`git push origin feature/AmazingFeature`)。
5.  **提交 Pull Request**：前往原專案頁面提交 Pull Request。

## 報告問題 (Issues)

如果您發現 Bug 或有功能請求，請使用 GitHub Issues 進行報告。請盡量提供詳細的資訊，包括：

*   問題描述
*   重現步驟
*   預期結果與實際結果
*   錯誤截圖或 Log（若適用）

## 開發規範

*   請保持程式碼整潔。
*   如果修改了 Python 腳本，請確保能夠在 Python 3 環境下正常執行。
*   提交訊息 (Commit Message) 請簡潔明瞭。

感謝您的支持！
